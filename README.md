# ⏰ Time & Age Calculator

A simple and responsive web application that displays the current time, date, and allows users to calculate their age based on their date of birth.

![Screenshot 2026-03-03 230722](https://github.com/user-attachments/assets/a7c4c38c-d3c3-49be-9228-0a84c95ad871)

## 🚀 Features

- **Real-Time Clock**: Displays hours, minutes, and seconds.
- **Date Details**: Shows the current day, date, month, and year.
- **Age Calculator**: Calculates exact age in years, months, and days.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices.
- **Modern UI**: Clean glassmorphism design with a gradient background.

## 🛠️ Technologies Used

- **HTML5**: Structure of the webpage.
- **CSS3**: Styling and animations.
- **JavaScript (ES6)**: Logic for time and age calculation.
